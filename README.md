# 전기차 충전소 통함 정보 시스템
 - 공공 API를 활용한 전기차 충전소 통합 정보 시스템
    1. 충전소 검색기능
        - 주소 검색 기반으로 충전소 목록 조회, 지역별 충전소 개수, 급속/완속 비율 등 차트 표현
    2. 지도기반 시각화, 주변 편의시설
        - 위도·경도 데이터를 활용한 마커 표시
    3. 충전소 상세정보 조회
        - 선택한 충전소의 충전기 타입, 수, 요금, 사업자 정보, 현재충전 여부, 예상 대기시간 출력
    4. 요금 비교 기능
        - 사업자별 평균 요금 및 금속/완속별 평균 요금 비교
    5. 경로 기반 추천
        - 출발지~도착지 경로 상 충전소 추천
